## Hooks
  - [Introducing Hooks](https://reactjs.org/docs/hooks-intro.html)
  - [useHooks](https://usehooks.com/)
  - https://velog.io/@velopert/react-hooks

### [useState](https://reactjs.org/docs/hooks-reference.html#usestate) returns a stateful value, and a function to update it
  - [Lazy initial state](https://reactjs.org/docs/hooks-reference.html#lazy-initial-state)
  - https://github.com/revirth/what-is-hooks/blob/8f8f8b94353a512de2589ed137b532202b0ee942/tutorial/src/App.js#L11

### [useEffect](https://reactjs.org/docs/hooks-reference.html#useeffect) will run after the render is committed to the screen
  - https://github.com/revirth/what-is-hooks/blob/8f8f8b94353a512de2589ed137b532202b0ee942/tutorial/src/Info.js#L11

### [useContext](https://reactjs.org/docs/hooks-reference.html#usecontext) accepts a context object (the value returned from React.createContext) and returns the current context value for that context

### [useReducer](https://reactjs.org/docs/hooks-reference.html#usereducer) accepts a reducer of type `(state, action) => newState`, and returns the current state paired with a dispatch method
  - https://github.com/revirth/what-is-hooks/blob/8f8f8b94353a512de2589ed137b532202b0ee942/tutorial/src/Counter.Reducer.js#L21

### [useMemo](https://reactjs.org/docs/hooks-reference.html#usememo) will only recompute the memoized value when one of the dependencies has changed
  - https://github.com/revirth/what-is-hooks/blob/8f8f8b94353a512de2589ed137b532202b0ee942/tutorial/src/Average.js#L28

### [useCallback](https://reactjs.org/docs/hooks-reference.html#usecallback) will return a memoized version of the callback that only changes if one of the dependencies has changed
  - https://github.com/revirth/what-is-hooks/blob/8f8f8b94353a512de2589ed137b532202b0ee942/tutorial/src/Average.Callback.js#L17

### [useRef](https://reactjs.org/docs/hooks-reference.html#useref)
  - https://github.com/revirth/what-is-hooks/blob/8f8f8b94353a512de2589ed137b532202b0ee942/tutorial/src/Average.Callback.js#L14
